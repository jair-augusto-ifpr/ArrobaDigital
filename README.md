# ArrobaDigital
ArrobaDigital é uma aplicação que utiliza Inteligência Artificial e visão computacional para estimar o peso de bovinos a partir de imagens ou vídeos. A tecnologia permite “pesar o boi” sem necessidade de balança física, trazendo mais agilidade, praticidade e eficiência para pecuaristas no manejo, controle do rebanho e tomada de decisões na pecuária
